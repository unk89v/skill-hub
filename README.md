# Skill Hub - AI 技能知识库

一个简洁优雅的 AI 技能导航平台，帮助你快速发现和了解各种 AI 技能。

## 功能特性

- 📂 **分类浏览** - 12 大技能分类，清晰明了
- 🔍 **智能搜索** - 支持名称、描述、标签多维度搜索
- 📊 **详情展示** - 使用统计、标签、关联技能
- 🎨 **简洁界面** - 三栏布局，一目了然

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 访问
open http://localhost:3000
```

## 技术栈

- Node.js + Express
- 原生 JavaScript
- Font Awesome Icons
